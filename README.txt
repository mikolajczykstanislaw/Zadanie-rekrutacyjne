Instrukcja dla kandydata na Frontend Developera
=============================================

1. Zainstaluj zależności projektu:
   ```
   npm install
   ```

2. Uruchom tryb obserwacji, aby automatycznie generować pliki podczas pracy:
   ```
   npm run watch
   ```

3. Zaimplementuj projekt na podstawie makiety z Figma:
   - Edytuj plik `index.html` (główna struktura strony)
   - Kod SCSS dodawaj w katalogu `/assets/scss`
   - Kod JavaScript dodawaj w katalogu `/assets/js`
   - Możesz instalować dodatkowe paczki przez `npm install`, jeśli ułatwią Ci pracę i nie wpływają negatywnie na działanie strony

4. Przeczytaj komentarze w projekcie Figma – zawierają wskazówki dotyczące implementacji każdej sekcji.
